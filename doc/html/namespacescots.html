Typedef Documentation

◆ abs_type

using scots::abs_type = typedef std::uint32_t

abs_type defines type of abstract states (default = uint32_t)

It is required to be an integer type. It determines implicitly an upper bound on the number of abstract states (default = 2^32-1).
