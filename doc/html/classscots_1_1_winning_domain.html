Detailed Description

The abstract states from which the controller is winning.

The WinningDomain is computed by one of the game solving algorithms in GameSovler.hh and represents the set of abstract states in {0,...,N-1} from which the controller is winning.

For a mathematical definition see the manual.pdf.
